# 중고차 가격 예측 프로젝트

### 🚗프로젝트 소개
중고차의 기본 정보와 추가적인 이력 정보를 포함하여 중고차 가격 예측에 적합한 요인들을 선정하여 중고차 거래에서 발생하는 정보 비대칭성 문제를 완화하고 합리적인 중고차 가격 결정 요인 분석의 방향성을 제시하고자 프로젝트를 진행하였다.

이용 사이트 : [Carmax](https://www.carmax.com/cars)
